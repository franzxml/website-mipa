# Website Fakultas MIPA UNTAN

## Deskripsi
Proyek ini adalah *landing page* untuk profil Fakultas Matematika dan Ilmu Pengetahuan Alam Universitas Tanjungpura. Website dirancang sebagai halaman informasi yang menampilkan program studi, fasilitas, kegiatan akademik, serta kontak fakultas. Tampilan dibuat sederhana, cepat, dan responsif tanpa menggunakan *framework* eksternal.

## Teknologi
- HTML5 untuk struktur halaman
- CSS3 untuk gaya dan tata letak
- *Vanilla JavaScript* untuk interaksi antar elemen
- Aset gambar lokal tanpa *CDN*

## Struktur Folder
```
root/
│── index.html

└── assets/

    ├── js/

    └── img/
        ├── logo-untan.png
        ├── riset-icon.webp
        ├── digital-brain.png
        ├── dosen-icon.avif
        ├── fasilitas-icon.jpeg
        └── lab-mipa.jpg
```

## Cara Menjalankan
1. Unduh atau *clone* proyek
2. Buka folder proyek
3. Jalankan file `index.html` di browser
4. Jika ingin *live reload*, gunakan ekstensi *Live Server* di VS Code
