# Website Profil Fakultas MIPA UNTAN

Repositori ini berisi *source code* untuk *landing page* profil Fakultas Matematika dan Ilmu Pengetahuan Alam (MIPA), Universitas Tanjungpura. Proyek ini dikembangkan menggunakan teknologi web standar tanpa *framework* CSS/JS eksternal untuk menjaga performa yang ringan dan struktur kode yang bersih.

## Deskripsi Proyek

*Website* ini dirancang untuk memberikan informasi esensial mengenai fakultas, termasuk program studi, fasilitas, informasi akademik, dan kontak. Desain antarmuka (*User Interface*) mengadopsi identitas visual Universitas Tanjungpura dengan dominasi warna kuning dan biru *navy*, serta tata letak yang *responsive* untuk berbagai perangkat.

## Fitur Utama

* **Responsive Design:** Tampilan menyesuaikan secara otomatis pada perangkat *desktop*, tablet, dan *mobile*.
* **Sticky Navbar:** Navigasi tetap terlihat di bagian atas layar dengan efek *shrink* dan perubahan warna indikator aktif saat pengguna melakukan *scrolling*.
* **Smooth Scroll:** Perpindahan antar bagian (*section*) dilakukan dengan animasi halus.
* **Interactive Elements:** Termasuk formulir kontak fungsional (simulasi *front-end*) dan penyematan peta lokasi (*embedded map*).

## Teknologi yang Digunakan

Proyek ini dibangun menggunakan:

* **HTML5:** Struktur semantik dokumen.
* **CSS3:** Penataan gaya, *Flexbox*, *Grid System*, dan animasi transisi.
* **Vanilla JavaScript (ES6):** Logika interaksi DOM, *event handling*, dan *Intersection Observer API*.

## Struktur Folder

Berikut adalah struktur direktori dalam repositori ini:

- index.html          (Halaman utama)
- assets/
  - css/
    - styles.css      (Lembar gaya utama)
  - js/
    - main.js         (Logika JavaScript)
  - img/              (Aset gambar dan ikon)

## Cara Menjalankan (Local Development)

Untuk menjalankan proyek ini di komputer lokal Anda:

1.  Lakukan *clone* repositori ini ke direktori lokal:
    `git clone https://github.com/username-anda/nama-repo.git`
2.  Buka folder proyek.
3.  Jalankan berkas `index.html` menggunakan peramban web (*web browser*) modern seperti Chrome, Firefox, atau Edge.
4.  Disarankan menggunakan ekstensi "Live Server" jika Anda menggunakan VS Code untuk pengalaman pengembangan yang lebih baik.
